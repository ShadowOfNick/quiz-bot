# 🤖 Smart Quiz Bot (Telegram)

Это продвинутый Telegram-бот для групповых чатов, сочетающий в себе функции игрового мастера, личного ассистента и секретаря. Бот работает на базе **Google Gemini (Gemma-3)** и обладает способностью понимать контекст переписки, создавая релевантные викторины, напоминая о важных делах и помогая организовывать встречи.

## ✨ Основные функции

- **🎯 Умные викторины (Quizzes)**: Бот анализирует последние сообщения в чате и генерирует вопросы на темы, которые обсуждали участники. Имеет систему автоматического завершения через 30 секунд.
- **⏰ Натуральные напоминания**: Бот понимает человеческую речь. Просто скажите: *"Бот, напомни нам завтра в 14:00 про встречу"*, и он создаст задачу в базе данных и тегнет вас в нужное время.
- **🤝 Координация встреч**: При обнаружении обсуждения планов бот предложит создать опрос для выбора удобного времени или места.
- **📊 Таблица лидеров**: Статистика каждого участника по результатам квизов доступна по команде `/quiz_scores`.
- **🖇 Сбор ссылок**: Бот автоматически сохраняет все ссылки, присылаемые в чат, для удобного поиска в будущем.
- **📝 Саммари чата**: Если в группе возник спор или долгое обсуждение, бот может кратко пересказать суть (функция накопления контекста (`MessageCollector`)).

## 🛠 Технологический стек

- **Core**: Python 3.12+ / Aiogram 3.x (Асинхронный фреймворк для Telegram)
- **AI Engine**: Google GenAI SDK (Model: `gemma-3-27b-it` или аналоги)
- **Database**: SQLite + `aiosqlite` (Асинхронная БД)
- **Scheduling**: `apscheduler` (Планировщик фоновых задач для напоминаний и таймеров)
- **Configuration**: `pydantic-settings` (Типизированные настройки из `.env`)

## ⚡ Особенности реализации

### 1. Поддержка моделей Gemma-3
Бот оптимизирован для работы с моделями Gemma, которые не поддерживают нативный `JSON mode`. Реализован кастомный парсер `parse_json_from_llm`, который надежно извлекает структурированные данные из текстовых ответов ИИ.

### 2. Приоритетное взаимодействие (Direct Triggers)
Бот использует систему приоритетов:
- **Прямое упоминание или Reply**: Бот отвечает мгновенно, показывая статус *"🤔 Секунду..."*. Эти ответы **не ограничены** дневными лимитами проактивности.
- **Проактивность**: Бот может сам влезть в разговор с интересным вопросом или напоминанием, если чат долго молчит или наоборот — очень активен. Этот режим ограничен лимитом (по умолчанию 20 сообщений в день), чтобы не быть навязчивым.

### 3. Автоматика квизов
В отличие от стандартных квизов, здесь внедрена логика `wait_and_finish_quiz`. Даже если в группе мало людей и не все нажали на кнопки, бот принудительно подведет итоги по таймеру, чтобы не "замораживать" чат.

## 📂 Структура проекта

```text
app/
├── db/                # Репозитории и модели данных
│   ├── repositories/  # Прямая работа с SQL-запросами
│   └── database.py    # Инициализация схемы (миграции)
├── handlers/          # Обработчики Telegram (Commands, Callbacks)
│   ├── group_messages.py # Логика ИИ в чатах
│   └── quiz.py        # Управление викторинами
├── llm/               # Интеграция с ИИ (Gemini/Gemma Providers)
│   └── prompts.py     # Системные инструкции (Persona)
├── services/          # Бизнес-логика (Analyzer, QuizService, ReminderService)
├── scheduler/         # Фоновые задачи (напоминания и проверки)
└── utils/             # Текстовые утилиты и буфер сообщений
```

## 🚀 Быстрый запуск

1. **Клонируйте репозиторий и создайте окружение**:
   ```bash
   python3 -m venv .venv && source .venv/bin/activate
   pip install -r requirements.txt
   ```

2. **Настройте `.env`**:
   ```env
   BOT__TOKEN=your_telegram_bot_token
   LLM__PROVIDER=gemma
   LLM__GEMMA_API_KEY=your_google_ai_studio_key
   LLM__GEMMA_MODEL=gemma-3-27b-it
   # Администраторы (через запятую)
   BOT__ADMIN_IDS=123456,789101
   ```

3. **Запустите бота**:
   ```bash
   python3 bot.py
   ```
